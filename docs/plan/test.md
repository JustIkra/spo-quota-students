  ├───────────────┼────────────────────────────────────────────────────┤
  │ Студенты      │ Записать студента (ФИО + номер аттестата), удалить │
  ├───────────────┼────────────────────────────────────────────────────┤

  └───────────────┴────────────────────────────────────────────────────┘
  ---
  Ключевые правила

  - Номер аттестата глобально уникален — один человек не может быть записан дважды в системе
  - Квота по умолчанию — применяется при создании новой специальности
  - Логин оператора — генерируется автоматически из названия СПО (транслитерация)

  ---
  Тестирование в браузере

  Открыть: http://localhost:9010

  Сценарий 1: Работа администратора

  1. ВХОД
     - Логин: admin
     - Пароль: admin123
     → Попадаем на панель администратора

  2. СОЗДАНИЕ СПО
     - Меню → "СПО"
     - Кнопка "Добавить СПО"
     - Ввести: "Политехнический колледж №1"
     - Сохранить
     → СПО появится в списке

  3. СОЗДАНИЕ ОПЕРАТОРА
     - Меню → "Операторы"
     - Кнопка "Добавить оператора"
     - Выбрать СПО: "Политехнический колледж №1"
     - Создать
     → Появится окно с логином и паролем — СОХРАНИТЬ!
     Пример: login=politehnicheskiy_ko, password=Xy7kL9mNpQ2w

  4. НАСТРОЙКА КВОТ
     - Меню → "Настройка квот"
     - Базовая квота: ввести 30, сохранить
     → Новые специальности будут создаваться с квотой 30

  5. ВЫХОД
     - Кнопка выхода в правом верхнем углу

  Сценарий 2: Работа оператора

  1. ВХОД
     - Логин: [сгенерированный логин оператора]
     - Пароль: [сгенерированный пароль]
     → Попадаем на панель оператора

  2. ДОБАВЛЕНИЕ СПЕЦИАЛЬНОСТИ
     - Меню → "Специальности"
     - Кнопка "Добавить специальность"
     - Ввести: "09.02.07 Информационные системы"
     - Сохранить
     → Специальность появится с квотой 30 (базовая)

  3. ЗАПИСЬ СТУДЕНТА
     - Меню → "Студенты"
     - Кнопка "Добавить студента"
     - Выбрать специальность: "09.02.07 Информационные системы"
     - ФИО: "Иванов Иван Иванович"
     - Номер аттестата: "123456789012"
     - Сохранить
     → Студент появится в списке

  4. ПРОВЕРКА УНИКАЛЬНОСТИ АТТЕСТАТА
     - Попробовать добавить ещё одного студента
     - Номер аттестата: "123456789012" (тот же)
     → Ошибка: "Номер аттестата уже зарегистрирован"

  5. СТАТИСТИКА
     - Меню → "Статистика"
     → Таблица: Специальность | Квота: 30 | Записано: 1

  Сценарий 3: Изменение квоты администратором

  1. Войти как admin

  2. Меню → "Настройка квот"
     → Таблица специальностей с текущими квотами

  3. Найти "09.02.07 Информационные системы"
     - Кнопка "Изменить"
     - Ввести новую квоту: 50
     - Сохранить
     → Квота изменена на 50

  4. Проверить в статистике
     → Квота теперь 50
